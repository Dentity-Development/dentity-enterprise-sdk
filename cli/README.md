# Okapi CLI

## Installation

### Using Homebrew

```
brew tap trinsic-id/trinsic
brew install trinsic
```

### Using Cargo

Requires [Rustup](https://www.rust-lang.org/tools/install) nightly toolchain

```bash
cargo +nightly install --git https://github.com/trinsic-id/sdk trinsic
```

## Usage

```
trinsic --help
```
